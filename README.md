# 📊 Flipkart Sales Dashboard

1.📊 Project Description

This project presents an interactive sales analytics dashboard built using Power BI. The dashboard helps visualize important patterns and insights from the dataset in a clear and simple way. It allows users to quickly understand trends, compare performance, and explore data through easy-to-read visuals.

2.⚙️ Tech Stack

📊 Power BI Desktop

📑 Data Visualization

📈 Dashboard & Report Design

3.🌐 Data Source

The dataset used in this project is obtained from the National Data & Analytics Platform (NDAP) by NITI Aayog.

🔗 https://ndap.niti.gov.in/

NDAP provides open government datasets that help in research, analytics, and policy insights.

4.📈 Charts Used in the Dashboard

📊 Bar Chart
Used to compare values across different categories and highlight differences in performance.

📉 Line Chart
Displays trends and changes over time, helping to observe patterns or growth in the dataset.

🥧 Pie Chart / Donut Chart
Shows the proportion or percentage distribution of different categories in the dataset.

🗺️ Map Visualization (if used)
Helps visualize geographic distribution of the data across different regions or states.

🔢 KPI / Card Visuals
Displays important summary numbers such as totals or key indicators in a clear and simple format.

📋 Table / Matrix
Presents detailed data in an organized format for deeper analysis.

🎛️ Slicers / Filters
Allow users to interact with the dashboard and filter data based on selected categories.

5.✨ Features / Highlights

📊 Interactive and easy-to-understand dashboard

📈 Visual representation of key insights

🔍 Quick comparison using charts and filters

🎯 Clean and simple layout for better understanding

📑 Data-driven insights in one place
