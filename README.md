# 📊 Flipkart Sales Dashboard – Power BI Project
This project showcases an interactive Sales Dashboard built using Power BI Desktop to analyze and visualize Flipkart sales data in a simple and insightful way. The dashboard helps understand sales trends, product performance, and customer behavior through clear visual charts and reports.
✨ Key Highlights

📈 Easy-to-understand sales insights

🛍️ Product and category performance overview

🌍 Region-wise sales analysis

📊 Interactive visual charts and filters

🎯 Clean and simple dashboard layout
💡 Why This Project?

Data becomes powerful when it tells a story. This dashboard helps turn numbers into insights that anyone can understand at a glance.
🚀 How to View

Download the .pbix file

Open it using Power BI Desktop

Explore the interactive dashboard
